🚀 Claude Code 一键安装指南

硅基API · 无需 Anthropic 账号 · 快速上手

一、一键安装命令

Linux / MacOS
Windows
curl -fsSL https://raw.githubusercontent.com/SamAISEO/GuijiAPI/main/install.sh | bash
irm https://raw.githubusercontent.com/SamAISEO/GuijiAPI/main/install.ps1 | iex

复制上面的脚本命令到命令行窗口，按回车键运行。

二、安装步骤说明

  • 获取 API Key
    访问 硅基API官网 注册并获取 API Key
  • 运行安装脚本
    在终端中粘贴并运行上面的安装命令
  • 输入 API Key
    根据提示输入 API Key（输入时不显示）
  • 选择模型
    选择默认使用的 Claude 模型

三、开始使用

安装完成后，关闭当前终端，重新打开一个新终端。
claude

输入 claude 命令启动 Claude Code。

首次启动提示：
选择 "1. Yes, I trust this folder" 确认使用配置的 API Key。
